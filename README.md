## 🧑‍💻 Projeto 

    Aplicativo mobile de receitas onde o usuário podera fazer cadastro e login, além de ver diversas receitas.
    O projeto foi desenvolvido ultilizando React Native com expo, firebase Database e Authentication.

 

### 📗 Rodando a Aplicação

```bash
📗 Installation

1. $ npm install 
  or $ yarn install
2. $ npx expo start
```
